Windows 实现 ZSH Git 命令缩写（alias）

# 背景

因为笔者经常 Win和 MAC 切换使用，两个平台提供的用户体验不同，但是本人的用户习惯已经养成了，有很多时候很不方便。

例如： 我想拉代码的时候会习惯性的使用缩写 = =

基于上面的原因我基于 ZSH Git 插件自带的 Alias 配置了一份类似的命令缩写。

特别提示: 本片文章仅适用于 Git 配置，如果你跟笔者一样使用 PowerShell 通过 posh-git 使用 Git 请看另一篇文章
Windows PowerShell Posh-Git 配置命令缩写（alias）

# 实现步骤

  1. 进入 Git 目录，找到配置文件
    D:\Git\etc\profile.d\aliases.sh
1
  1. 编辑 aliases.sh 文件
# Some good standards, which are not used if the user
# creates his/her own .bashrc/.bash_profile

# --show-control-chars: help showing Korean or accented characters
alias ls='ls -F --color=auto --show-control-chars'
alias ll='ls -l'
# --> 在这里追加新的配置

case "$TERM" in
xterm*)
	# The following programs are known to require a Win32 Console
	# for interactive usage, therefore let's launch them through winpty
	# when run inside `mintty`.
	for name in node ipython php php5 psql python2.7 winget
	do
		case "$(type -p "$name".exe 2>/dev/null)" in
		''|/usr/bin/*) continue;;
		esac
		alias $name="winpty $name.exe"
	done
	;;
esac
